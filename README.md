# ABDUL Registration platform

## Function Feature
1. Display API list (Public)
2. Google Login
3. Register API 
4. Display Own API && can Sort Pub. or Pri. && Lastest or Oldest
5. Manage ALL API for Superuser
6. Owner of API and Superuser can Delete and Fix API
